Course Description

This advanced course dives deep into Simultaneous Localization and Mapping (SLAM) techniques for autonomous robots. You'll learn how to equip robots with the ability to construct maps of unknown environments while simultaneously keeping track of their location within these maps.

Through hands-on projects with ROS (Robot Operating System) and real sensors, you'll implement various SLAM algorithms and navigation stacks. By the end, you'll be able to design robots capable of autonomous exploration and navigation in complex, dynamic environments.

Core SLAM Concepts

Mapping Techniques

Learn various approaches to environmental mapping including occupancy grids, feature maps, and topological maps.

  • Grid-based mapping
  • Feature extraction
  • Map optimization

Localization

Master probabilistic localization methods including Kalman Filters, Particle Filters, and Monte Carlo Localization.

  • Probabilistic robotics
  • Sensor fusion
  • Uncertainty modeling

Path Planning

Implement advanced path planning algorithms for autonomous navigation in dynamic environments.

  • A* and D* algorithms
  • Potential fields
  • Dynamic replanning

Autonomous Decision Making

Develop intelligent behaviors for exploration, obstacle avoidance, and goal-oriented navigation.

  • Behavior trees
  • State machines
  • Reinforcement learning

SLAM Algorithm Walkthrough

gmapping Process

1

Laser Scan Data Acquisition

The robot collects laser scan data from its environment using LIDAR sensors, measuring distances to surrounding objects.

2

Odometry Integration

Wheel encoder data is combined with IMU readings to estimate the robot's movement between scans.

3

Scan Matching

Consecutive laser scans are aligned to correct odometry drift and improve position estimation.

4

Map Update

The algorithm updates the occupancy grid map with new scan data, marking obstacles and free space.
